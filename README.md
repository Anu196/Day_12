# Day_12
This AI WhatsApp Chatbot is built using n8n, CloudStation Evolution API, and Google Gemini AI. It receives WhatsApp messages, sends them to the AI model, generates an intelligent response, and automatically replies to the user on WhatsApp in real time.

## **AI WhatsApp Chatbot (Day-12)**

### **Project Description**

The **AI WhatsApp Chatbot** is an **n8n-based automation workflow** that provides intelligent, real-time responses to WhatsApp messages. It integrates **CloudStation Evolution API** with **Google Gemini AI** to receive user messages, process them using AI, and automatically send relevant replies back on WhatsApp. The chatbot enables businesses and individuals to automate conversations, provide instant support, and improve communication efficiency.

---

## **Features**

* 💬 Receives WhatsApp messages in real time
* 🤖 Generates intelligent responses using Google Gemini AI
* 🔄 Automatically replies to users through the CloudStation Evolution API
* ⚡ Fully automated workflow built with n8n
* 📱 Real-time two-way WhatsApp communication
* 📝 Handles a wide variety of user queries
* 🔒 Secure API integration for reliable messaging
* 🚀 Fast response generation with minimal delay
* 📊 Easy integration with databases or Google Sheets for conversation logging
* 🌐 Scalable solution for customer support and AI assistance

## **What the Project Does**

* Receives messages sent by users on WhatsApp.
* Sends the message to Google Gemini AI for processing.
* Generates an intelligent and context-aware response.
* Automatically delivers the AI-generated reply back to the user on WhatsApp.
* Provides instant, automated communication without manual intervention.
